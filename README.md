# ms-spring-sec
Micro service for Spring Security

## Tech Stack

**Java:** version 17

**Spring:** 3.2.2

**OpenApi:** V3

## Installation

Install ms-spring-sec with maven

```bash
  mvn clean install ms-spring-sec
```


## Deployment

To deploy this project run

```bash
  mvn spring-boot:run
```
